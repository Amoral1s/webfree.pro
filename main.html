Какими языками владею и пользуюсь?

  • html + css - В основном пользуюсь разметкой bootstrap4 для удобного позиционирования обьектов и адаптации под устройства с любыми размерами экранов.
  • JavaScript - препроцессор jQuery - самая популярная и быстрая библиотека анимации и отзывчивости для вашего сайта. Так же при ваших потребностях, могу написать на чистом JS.
  • PhP - Создание форм отправок email. Остальные функции PhP использую в WordPress для гибкости контента.
  • CMS система WordPress - Очень популярная система управления вашим сайтом, позволяет менять вам содержимое вашего сайта самостоятельно. Возможность ввести регистрацию пользователей, вплоть до создания интерент-магазина итд.
  • gulp+sass - Позволяет в разы ускорить создание проекта и очистки от ненужного мусора на стадии передачи его вам. А так же упрощает в дальнейшем редактирование кода другими программистами.
  • Мои знания с каждым днем увеличиваются, в дальнейшем этот список расширится...
